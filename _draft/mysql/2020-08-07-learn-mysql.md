# 索引
索引是一种数据结构，用于帮助我们在大量数据中快速定位我们想要查找的数据。索引在mysql中分为三类:
- B+树索引
- Hash索引
- 全文索引
Innodb存储引擎中的B+树索引，B+树是从二叉查找树、平衡二叉树、B树这三种结构演化而来。
